# Problem 3

This folder contains three implementations for Problem 3.

## Files

- `problem3_main - explicit.py`  
  Runs the explicit quantum model.

- `problem3_main - implicit.py`  
  Runs the variational implicit quantum model.

- `problem3_main - reupload.py`  
  Runs the serial data re-uploading quantum model.

## How to run

Please use Python 3.

```bash
python3 "problem3_main - explicit.py"
python3 "problem3_main - implicit.py"
python3 "problem3_main - reupload.py"
```

## Requirements

Please install the required Python packages before execution.  
Typical packages include:

- Python 3.x
- PennyLane
- torch
- torchvision
- numpy
- matplotlib

## Notes

- Each script can be executed independently.
- These three scripts correspond to the three different Problem 3 model settings discussed in the report.
- Please refer to the submitted PDF report for experimental results and discussion.
